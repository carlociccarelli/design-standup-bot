Installing.

Local distribution, not from the App Store. Here's what to expect.

  1. Download DesignStandupBot.dmg from the button above.
  2. Double-click to mount. Drag the app into /Applications.
  3. First launch: macOS may show "DesignStandupBot can't be opened because it is from an unidentified developer" — this app isn't signed by Apple. To bypass: right-click the app → Open → confirm.
  4. The teal app icon appears in your menu bar. Click it.
  5. Tap Figma, Jira, GitHub, or Trello on the onboarding screen and follow the setup.

The Gatekeeper warning is normal for self-distributed Mac apps. Once you've opened it once, macOS remembers your choice — subsequent launches happen without prompt.

Questions.

How do I get a Figma Personal Access Token?

In Figma: click your avatar (top right) → Settings → Security → Personal access tokens → Generate new token. When prompted for scopes, grant at minimum: File content (read-only) and Team library content (read-only). Paste the token into Design Standup Bot's Settings.

Where do I find my Figma Team ID?

Open your Figma team in the browser. The URL looks like figma.com/files/team/849554081353053086/... — the long number is your Team ID. Alternatively, leave Team ID empty and paste individual file URLs in the "File keys" field — works for any Figma user.

How do I get a Jira API token?

Visit id.atlassian.com/manage-profile/security/api-tokens → Create API token. The base URL is your Atlassian subdomain, e.g. https://your-team.atlassian.net. Your email is the one you sign in to Atlassian with.

Will I need to be a Jira admin?

No. The app uses a JQL that works for any Jira user: assignee = currentUser() OR (watcher = currentUser() AND updated >= -7d). No board membership or admin permissions required.

How do I get a GitHub Personal Access Token?

On GitHub: click your avatar (top right) → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token. Grant the repo scope (read-only is fine; we never write back). For GitHub Enterprise, paste your API base URL (e.g. https://github.acme.com/api/v3) in the optional Enterprise URL field. You can also limit the digest to specific repos by entering owner/repo one per line.

How do I get a Trello API Key and Token?

On a single page at trello.com/app-key: copy the API Key at the top, then click the "Token" link below it to generate a user Token (one click, grants read access). Paste both into Design Standup Bot's Settings. Read-only is fine — the app never writes back to Trello.

Why does macOS keep asking me for my login password?

If you see repeated Keychain prompts during development builds, that's because the code-signing identity changes between builds, and macOS treats each new signature as a different app requesting Keychain access. In the distributed DMG (single signature, stable), the prompt happens once at first launch — pick "Always Allow" and it won't ask again.
